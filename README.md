# Online Examination Management System

## Overview

This project was developed as part of a Software Engineering course.

The Online Examination Management System provides a secure platform for managing online examinations, supporting students, instructors, and administrators.

## Features

- User Authentication
- Role-Based Access
- Exam Creation
- Online Exam Interface
- Automatic Grading
- Manual Grading
- Result Generation
- AI-based Cheating Detection Concept
- Performance Reports

- Test Cases
